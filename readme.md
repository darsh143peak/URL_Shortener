# 🔗 Smart URL Shortener with Analytics

A full-stack web application that converts long URLs into short, shareable links and provides basic analytics such as click tracking and creation details.

---

## 🚀 Live Demo

👉 https://url-shortener-k0i0.onrender.com

---

## ✨ Features

- 🔗 Shorten long URLs into unique short links
- 🔁 Seamless redirection to original URLs
- 📊 Click tracking for each short link
- 📅 Creation date tracking
- ⏳ Optional expiry for links
- 🎯 Custom short URLs (user-defined aliases)
- 📋 Dashboard to view all links and analytics
- ⚠️ Handles invalid inputs gracefully
- 🔒 Ensures uniqueness of short URLs

---

## 🛠️ Tech Stack

- **Backend:** Flask (Python)
- **Database:** SQLite
- **Frontend:** HTML, Tailwind CSS
- **Deployment:** Render
- **Server:** Gunicorn
